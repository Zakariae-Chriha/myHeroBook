# App Icons Required

Place your app icon files here. All must be PNG with transparent or solid background.

| File          | Size       | Used by                      |
|---------------|------------|------------------------------|
| icon-72.png   | 72×72 px   | Android LDPI/MDPI            |
| icon-96.png   | 96×96 px   | Android MDPI                 |
| icon-128.png  | 128×128 px | Chrome Web Store             |
| icon-144.png  | 144×144 px | Android XHDPI / Windows tile |
| icon-152.png  | 152×152 px | iPad                         |
| icon-192.png  | 192×192 px | Android launcher (maskable)  |
| icon-384.png  | 384×384 px | Android splash               |
| icon-512.png  | 512×512 px | PWA / Play Store             |

## Design tip
Use dark navy background (#0A0E1A) with your gold star/hero logo centered.
The 192 and 512 icons should be "maskable" — keep the logo inside the safe zone (central 80%).

## Quick generator
Upload one 1024×1024 PNG to: https://maskable.app/editor
Then export all sizes at once.
